# Generative AI with Databricks

This repository contains my hands-on work from an instructor-led
Generative AI program using Databricks.

## What this repo contains
- Recreated Databricks notebooks from concepts learned
- Prompt engineering and LLM inference
- Embeddings and vector search
- Retrieval Augmented Generation (RAG) pipelines

## Tech Stack
- Databricks Community Edition
- PySpark
- MLflow
- Hugging Face / OpenAI APIs
- Streamlit

## Goal
To build an end-to-end GenAI application that demonstrates
production-style architecture and best practices.
